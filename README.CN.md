# Gudupao Open Source

[English](README.md)｜简体中文



## 简介

Gudupao Open Source是一个为需要贡献的代码仓库而设计的分享平台。

## 如何贡献本仓库（提交您的仓库）

首先您需要Fork本仓库，更改`data/project.json`文件，按以下格式：

```json
{
      "id": "请根据上一个id自加1",
      "name": "xxx",
      "description": "xxx",
      "github": "https://github.com/xxx/xxx",
      "tags": ["xxx", "xxx"],
      "stars": 0,
      "forks": 0
    },
```

如果您不会操作请联系[official@gudupao.top](mailto:official@gudupao.top)
